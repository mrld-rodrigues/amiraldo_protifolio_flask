ANDROID APP - APP_GEO_LOC

APP_GEO_LOC is a lightweight Android application developed using Python, powered by Kivy and KivyMD, designed to retrieve and display the real-time geolocation of the user’s device on a map interface. Built with mobile simplicity and cross-platform flexibility in mind, the app fetches GPS coordinates and marks the current location visually with a pin on an interactive map, providing a straightforward way to track positioning on the go. Built with Python 3 and Kivy for the app’s structure and UI, using KivyMD to provide a polished Material Design look. It accesses Android’s GPS services to get real-time location data and displays a marker on a map via a map API. Packaged as an Android (.apk) app, it’s optimized for smooth touch interaction and performs well on low- to mid-range devices. This is a basic GPS location tracking, personal navigation, proof-of-concept for location-based apps, or as a foundation for more complex geolocation services.

VIDEOGAME HANGMAN- JOGO DA FORCA

Hangman is a modern reinterpretation of the classic hangman game, built from scratch in Python using the Pygame library. More than just guessing words, the game offers a dynamic experience with four progressive difficulty levels, ranging from simple and common words to complex and rare terms, with adaptive attempt limitations. Built entirely with Python 3 and Pygame for graphics rendering, event handling, and game state management. The structure is organized in modules to facilitate maintenance and expansion (game loop, level control, word system, UI). Simple and functional graphical interface, developed and tested exclusively on Linux – not yet compiled to .exe (Windows).

IEFP- FINAL COURSE PROJECT
